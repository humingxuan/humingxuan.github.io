## 删除线

````
~~markdown~~
````

显示效果：

~~markdown~~

---

## 下划线

```
<u>文本</u>
```

显示效果：

<u>文本</u>

---

## 脚注

```
我在xxx[^runoob]学习Markdown语法

[^runoob]:xxx
```

显示效果：

我在xxx[^runoob]学习Markdown语法

[^runoob]:xxx

在文档中脚注会==自动编号==，脚注出现在文档的最后。

## 使用HTML元素

Markdown语法支持的HTML元素有：\<kbd>\<b>\<i>\<em>\<sup>\<br>

```html
<kbd>Ctrl</kbd>键
```

显示效果：

<kbd>Ctrl</kbd>键

## 复制

### 复制普通文本

​	<kbd>ctrl</kbd>+<kbd>c</kbd>

### 复制Markdown文本

​	<kbd>ctrl</kbd>+<kbd>shift</kbd>+<kbd>c</kbd>

---

## 公式

### 行间公式

```markdown
$$ 公式
```

显示效果：
$$
y=x
$$

### 行内公式

```
$公式$
```

显示效果：

$y$与$x$的关系为$y=x$

不过，这里有一点需要注意：

行内公式为LaTeX格式，Typora默认情况下是不开启的，需要在**偏好设置**中点选相关选项
